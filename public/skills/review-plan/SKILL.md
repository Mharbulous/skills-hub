# Review Plan

Review a plan for flaws and opportunities for improvement, then write an improved plan that incorporates everything good about the original while correcting flaws and adding improvements.

## Input

The plan to review is provided as arguments: `$ARGUMENTS`

## Output Structure

1. **Analysis** -- Briefly list the flaws found and improvements identified
2. **Improved Plan** -- The complete rewritten plan
