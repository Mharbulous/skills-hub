# Most Popular Universal Tips for Global CLAUDE.md

*Compiled 2026-04-19 via sequential deep-research subagents.*

This table catalogs recommendations for content to include in `~/.claude/CLAUDE.md` (the user-level global CLAUDE.md loaded for every session). Each entry is a phrase recommended by a public source, with the date of the source and a URL citation. Caveats, exceptions, and side effects are included where the source addressed them.

**Why date matters:** Claude model capabilities have shifted substantially over the past year (Sonnet 3.5 → 4 → 4.5 → 4.6; Opus 4 → 4.5 → 4.7; Haiku 3.5 → 4.5). Advice that was correct for older models may now be counterproductive. Prefer recent sources.
