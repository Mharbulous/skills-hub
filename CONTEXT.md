# Context

## Glossary

**Resolver Wrapper.** A Cowork package that verifies and materializes hosted
Skills-hub content at runtime before the agent reads the verified local
`SKILL.md`.

**Skills-hub Control Panel.** The verified `/skills-hub` skill for inventory,
install, and update of Cowork resolver wrappers.

**Degraded Inventory.** Local-only installed-wrapper evidence shown when catalog
verification or availability is blocked; it is never a basis for install or
update.
